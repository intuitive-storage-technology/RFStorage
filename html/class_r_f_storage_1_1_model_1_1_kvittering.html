static void RFStorage.Model.Kvittering.KvitteringTilbage ( )
static

Denne metode initialisere selve pdfdokumentet og memorystreamen og kalder til sidst Save-hjælpe metoden. Dette pdfdokumentet bruges i forbindelse med tilbagelevering

Exceptions : Vil smide en null-reference exception hvis TilbageleveringsOC peger på null og dette ikke er checket før klassen bliver kaldt

Pre-Conditions : UdleveringsOC skal eksistere, dette gøres i forbindelse med udlevering af varene

Post-Conditions : Efter metoden er kørt igennem, vil opsætningen af pdf-dokumentet være lavet og Save metoden kaldes.

Side-Effects : Der er ingen sideeffects ved denne metode
